# ugly-scratch

> sb3 코드 난독화

+ 변수, 리스트, 방송 이름 변경
+ 함수 제목 및 인수 이름 변경

## 사용법

```bash
$ git clone https://github.com/kangdongmandoo/ugly-scratch.git
$ cd ugly-scratch
$ npm install # yarn
$ npm run dev # yarn start
```

또는 [사이트](https://ugly-scratch.netlify.app/)를 이용하세요.  

파일을 선택한 후 기다리면 자동으로 난독화된 코드가 다운로드됩니다.
